## 🚀 Principais Projetos (Destaques)

### 📌 [Boilerplate Django Completo (Admin)](https://github.com/rogeriokostha/boilerplate-completo-admin)
Um boilerplate Django extremamente robusto para iniciar projetos rapidamente. Inclui:
*   Autenticação JWT, Django REST Framework (DRF) e Swagger.
*   Painel Administrativo modernizado com Jazzmin.
*   Estrutura escalável com localização e testes automatizados configurados.

### 📌 [BaseAdmin](https://github.com/rogeriokostha/baseadmin)
Uma arquitetura comercial e limpa em Django, projetada para aplicações de nível empresarial (SaaS, CRMs, ERPs).
*   Focado em segurança, métricas integradas e design moderno de painéis.
*   Tratamento de dados estruturados com contexto geográfico (Brasil) diretamente ligado ao core do sistema.
